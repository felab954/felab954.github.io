PHP之日志系统SeasLog安装

PHP之日志系统SeasLog安装

六月 20, 2017
  • mac
  • php

An effective,fast,stable log extension for PHP.

1、下载SeasLog wget -c https://github.com/SeasX/SeasLog/archive/SeasLog-1.6.9.tar.gz

2、解压下载好的文件包 tar -zxvf SeasLog-1.6.9.tar.gz

3、切换到SeasLog-1.6.9目录执行phpize命令，如果phpize命令找不到，那就用绝对路径执行/usr/local/php/bin/phpize操作，目录中会多出一个configure文件（需要安装autoconf依赖）

4、检测SeasLog是否依赖别的包，在当前文件执行 ./configure --with-php-config=/usr/bin/php-config

5、编译并安装 make && make install

6、在php.ini文件中配置SeasLog信息

如果安装后没有生效，检查PHP版本和SeasLog版本是否兼容。
